# Security Policy

Boustrophidon is a LaTeX package and does not intentionally execute external commands.

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |

## Reporting a Concern

Open a private report through GitHub Security Advisories if the repository enables them. Otherwise, contact the maintainer through the public repository issue tracker with enough detail to reproduce the concern.

## Scope

Relevant security concerns include:

- Unexpected shell escape behavior.
- Unsafe file access behavior.
- Malicious or compromised release artifacts.

Normal LaTeX compilation warnings, typography issues, and missing glyph behavior should be reported as regular issues.
